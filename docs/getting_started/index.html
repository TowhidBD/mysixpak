Getting Started

MySixPak HTML Template User Guide

Thank you for downloading our template! We really appreciate it and ready to help you!

MySixPak HTML Template is a large variety of layout and content options are available. It comes with HTML5 and Bootstrap 4.5. It comes with modern design and a faster loading HTML template. Below are directions and example code for the various elements of the MySixPak HTML template.

caution

Does your MySixPak download look incomplete? Try using either WinRAR or http://www.7-zip.org to open the zip!

Installing#

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the /HTML folder to find all the template files. You will need to upload these files to your hosting web server using FTP in order to use it on your website.
  2. Below is the folder structure and needs to be uploaded to your website root directory:
    • assets/css - Stylesheets folder
    • assets/fonts Icon fonts folder
    • assets/img - Images folder
    • assets/js - Javacripts folder
    • assets/vendor Vendors css/js folder
    • 404.html 404 not found
    • favicon.ico Favicon
    • icon.png Icon
    • index.html a default HTML skeleton that should form the basis of all pages on your website
    • style.css Theme css
  3. You should upload all or specific HTML files as per your need.
  4. You're now good to go..! Start adding your content/images and generate your beautiful brand new website for your awesome users.

File Uploading#

After you download your website template from Freelancer.com project files and make all of your edits you're going to need to upload the files to your web server. You will need an FTP client such as Dreamweaver or FileZilla. In your FTP client add your server details (get this from your web host provider) and then you can connect to your server and upload your files.

What's Included#

Once you download the template, you can see a folder with several files and sub folders. You can read description of some important files in continue.

  • assets
    • css
      • main.css Base styles: opinionated defaults
      • custom.css Write your custom styles in this file
      • msp-icon.css MySixPack icon css
      • normalize.css Normalize.css makes browsers render all elements more consistently.
      • responsive.css Responsive CSS makes your web page look good on all devices.
    • js
      • jquery-1.12.4.min.js jQuery Foundation
      • jquery-ui.min.js jQuery UI
      • main.js Main js file
      • plugins.js Avoid console errors in browsers that lack a console.
      • popper.min.js TOOLTIP & POPOVER POSITIONING ENGINE

    • img Image Folder
    • fonts Fonts Folder
    • vendor Vendor css/js Folder
